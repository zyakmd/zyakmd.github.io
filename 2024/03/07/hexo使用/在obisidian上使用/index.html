# 位置

  1. 关闭  使用 Wiki 链接
  2. 使用插件 Custom Attachment Location（里面还有自动为粘贴的图像重命名的功能）
  3. 在↑设置中的 Location xx：./$
  4. 设置 “文件与链接” 中附件选 “当前文件所在文件夹的子文件夹”，然后留空（由上面的插件自动填入了）
  5. 设置 “文件与链接” 中内部链接类型选 “尽可能短的方式”
