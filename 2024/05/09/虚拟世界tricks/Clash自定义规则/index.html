# 前言

情景：GPT 需要稳定的魔法，但这个魔法的自带规则⑧行，像国内我常用 Bing 搜索，规则里却没有直连，国际版有时候也会 bug。因此需要自定义规则，直接加在订阅的规则里会在更新订阅链接后被覆盖

# 解决

在 settings 的 Profiles 的 Parsers 中设置预处理器内容

在订阅配置文件触发刷新（包括自动更新）时，CFW 会读取该 yaml 字段定义的值，将对应值插入 / 合并到原配置文件中

此外，这里还可以自定义 proxies ，以及更多参数见官方文档

调教好就可以流畅的一直开魔法了
