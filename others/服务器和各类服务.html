# zookeeper

参考

根目录：usr/zookeeper/apa....-bin/
在 conf/zoo.cfg 配置

bin 目录 ./zkServer.sh start 启动 ... stop/... restart/... status
./zkServer.sh start-foreground 查看日志启动，debug

  • ps aux | grep zookeeper 查看当前进程 PID，只有一个就是查询的进程，第一个数字是 pid
  • kill -9 PID 关闭

存储在：tmp/zookeeper/zkdata
